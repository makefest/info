# Createing an Outlook Account

## Create Entry in your Password Manager

The steps below assume that you are using a keepass password manager such as KeePassXC. But the steps should be very similar for other password managers.

- 🔨 make sure you have a group called **Microsoft accounts**
- 🔨 in that group, create a new entry (⌨️: `Ctrl`+`N`)
- 🔨 for both the title and username, add the e-mail you **want**, ending in `outlook.com`
- 🔨 tab into the password field and generate a strong password (⌨️ `Ctrl`+`G`)
- 🔨 tab into the URL field and enter the following:

```markdown
https://outlook.com
```
- 🔨 save the entry (⌨️ `Ctrl`+`Enter`)

## Create Outlook Account

- 🔨 ⚠️ make sure, you do NOT have a VPN running! An active VPN may prevent the account creation from succeeding.
- 🔨 go to [outlook.com](https://outlook.com)
- 🔨 if you are signed in, sign out
- 🔨 make sure you are still on [outlook.com](https://outlook.com)
- 🔨 click on `Create free account`
- 🔨 make sure the domain `outlook.com` is chosen
- 🧠 the e-mail you would like may already be taken
- 🔨 repeat the following steps until you've requested an available e-mail address:
  -  _enter an e-mail name you would like to have_
  -  _click `Next`_
- 🔨 in case you had to chose another e-mail name, update the entry in your password manager - do it now, not later
- 🔨 when asked for the password, copy it from your password manager and paste it in
- 🔨 in the following dialogs, you will be asked a couple of questions; the choice is yours but here are my recommendations for how to answer them:
  - keep in mind that you might be creating the account for yourself or for someone else, like a kid of yours; so the info you provide is of course for the person for which you create this account
  - put in the actual information for the person's place and birthdate
  - put in the actual information for the person's name
  - uncheck the option to receive information
- 🦉 you will need to solve a [captcha](https://en.wikipedia.org/wiki/CAPTCHA) and this may take concentration and time - please be patient
- 🔨 solve the captcha
- 🔨if asked whether you want to "Stay signed in?", check the box for "Don't show this again" and click `No`
- 👁️ you should now be signed in

## Test signing in

- 🔨 go to [outlook.com](https://outlook.com)
- 🔨 if you are signed in, click on your profile picture in the top-right corner and sign out
- 🔨 make sure you are still on [outlook.com](https://outlook.com)
- 🔨 sign-in again, **copying your username and password from your password manager**
- 🦉 by creating an outlook email, you have automatically created a Microsoft account
